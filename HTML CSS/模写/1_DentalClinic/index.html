医院紹介

診察コンセプト

予防の大切さをしてもらう

毎月行っている「こどもデンタル教室」では、お子さんたちの虫歯や口膣内の疾患の予防を指導しています。

正しい歯磨きのやり方、しっかり噛むことでどのような効果があるか、進行した虫歯の恐ろしさなど... 大人の方にもぜひ聞いていただきたい内容です。

お子さんたちは真剣に内容に取り組んでくれています。検診の際に歯の一本一本をチェックして、 「お家で実践してくれている。保護者の方も協力してくださっているのだな」と成果を垣間見る事ができます。

こういった取り組みは、一時期のお子さんたちの虫歯の増加が発端になっていあます。

医院に通ってくださるお子さんや、検診で伺った小学校の児童さんの歯の状態が非常に良くなかったのです。 原因は様々だと思いますが、正しく予防さえできてればすぐに改善できるのはわかりました。 皆さん、知らないだけなのです。

以前は歯医者さんに対して「痛いことをされる怖いところ」だと思われていましたが、今では少し変わってきて 「痛くならないようにしてくれるところ」だと思っていただけているみたいです。

教室を初めて、顔を出してくださるお子さんは少しずつ増えてきました。今では翌月の予約がすぐに 埋まってしまうようなありがたい状態です。

一も二もなく、まずは予防することです。

虫歯になっても削ってしまえば済むことでは？と思われている方も少なく有りませんが、歯は大切な臓器です。 減らしたり失ったりすることは良いことではありません。

歯医者さんは、歯を大切にする方法を教えてくれるところに変わっていくべきなのです。

私達は、予防するという医療を地域に根付かせていきたいと日々努力しています。
